## Write a program to swap two numbers without using third variable.

```
Input:  a=10 b=20
Output: a=20 b=10
```

---

<CodeBlock slots="heading, code" repeat="2" languages="Python, C" />

#### Python

```python
# Write a program to swap two numbers.
# define variables
a = 10
b = 20

# swap numbers
c = a
a = b
b = c

# print result
print(a, b)
```
