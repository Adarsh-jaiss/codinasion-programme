## Write a programme to multiply two numbers.

```
Input:  5 4
Output: 20
```

---

<CodeBlock slots="heading, code" repeat="1" languages="Python" />

#### Python

```python
# input first number
a = int(input('Write the first number'))

# input second number
b = int(input('Write the second number'))

# output
print('The output is ', a*b)
```
